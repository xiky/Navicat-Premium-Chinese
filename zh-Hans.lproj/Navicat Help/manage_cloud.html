管理 Navicat Cloud

查看云使用率

  • 在主窗口的工具栏点击你的头像。

  • Navicat Cloud 窗口会显示你的云使用率和当前方案。

【注意】一个连接、查询、模型或虚拟组为 1 单位。

更改你的头像

  • 在主窗口的工具栏点击你的头像。

  • 在 Navicat Cloud 窗口中，点击头像。

  • 选择图像文件。

管理你的 Navicat Cloud 帐号

  • 在主窗口的工具栏点击你的头像。

  • 在 Navicat Cloud 窗口中，点击你的电子邮箱并选择“管理帐号”。

  • 浏览器将会打开 Navicat Cloud Portal 网站。

升级 Navicat Cloud 方案

  • 在主窗口的工具栏点击你的头像。

  • 在 Navicat Cloud 窗口中，点击“升级”。

  • 浏览器将会打开 Navicat Cloud Portal 网站。

创建一个项目

  • 在导航窗格中选择“Navicat Cloud”。

  • 按住 Control 键并点按它，然后选择“新建项目”。

添加成员到一个项目

  • 在导航窗格中选择一个项目。

  • 按住 Control 键并点按它，然后选择“协作与”。

  • 点击“添加成员”。

  • 输入成员的 Navicat ID。

  • 选择成员角色。

  • 点击“添加”。

成员角色

权限

拥有者

读取对象、写入对象、管理成员和删除项目

管理员

读取对象、写入对象和管理成员

成员

读取对象和写入对象

客人

读取对象

【注意】每一次可添加最多 10 个成员。在编辑框中使用逗号或输入键来分隔成员。

在一个项目管理成员

  • 在导航窗格中选择一个项目。

  • 按住 Control 键并点按它，然后选择“协作与”。

  • 更改后点击“应用”。

【注意】如果你是拥有者或管理员，你可以点击“x”按钮来移除成员。

重命名一个项目

  • 在导航窗格中选择一个项目。

  • 按住 Control 键并点按它，然后选择“重命名”。

  • 输入项目名。

退出一个项目

  • 在导航窗格中选择一个项目。

  • 按住 Control 键并点按它，然后选择“退出项目”。

删除一个项目

  • 在导航窗格中选择一个项目。

  • 按住 Control 键并点按它，然后选择“删除项目”。

移动或复制一个连接到 Navicat Cloud

  • 按住 Control 键并点按在“我的连接”的连接，然后选择“移动连接到”或“复制连接到”。

  • 选择现有的项目或创建一个新项目。

  • 连接会移动或复制到 Navicat Cloud。它的所有查询和虚拟组亦会保存在 Navicat Cloud。

移动或复制一个连接到我的连接

  • 按住 Control 键并点按在“Navicat Cloud”的连接，然后选择“移动连接到”或“复制连接到”->“我的连接”。

  • 连接会移动或复制到我的连接。它的所有查询和虚拟组亦会保存在本地机器中。

移动或复制一个模型到 Navicat Cloud

  • 在“我的连接”选择一个模型文件。

  • 拖放它到“Navicat Cloud”中的一个项目。

移动或复制一个模型到我的连接

  • 在“Navicat Cloud”的项目内选择一个模型文件。

  • 拖放它到“我的连接”。
