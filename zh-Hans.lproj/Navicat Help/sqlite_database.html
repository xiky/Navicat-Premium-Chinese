数据库

若要开始使用服务器对象，你应该创建并打开一个连接。在连接窗口常规选项卡中设置的数据库文件名为“main”数据库。

附加一个数据库文件

  • 在导航窗格中，按住 Control 键并点按连接，然后选择“附加数据库”。

  • 在弹出的窗口中输入数据库属性。

分离一个数据库

  • 在导航窗格中，按住 Control 键并点按已附加的数据库，然后选择“分离数据库”。

加密 main 数据库

  • 在导航窗格中，按住 Control 键并点按 main 数据库，然后选择“加密数据库”。

  • 在弹出的窗口中输入密码。

解密 main 数据库

  • 在导航窗格中，按住 Control 键并点按 main 数据库，然后选择“解密数据库”。

  • 在对话窗口中确认解密。

查看 sqlite_master 表

  • 在导航窗格中，按住 Control 键并点按数据库，然后选择“查看主表”。

  • sqlite_master 表打开于数据查看器中。
